# ❓ FAQ - Questions Fréquemment Posées

## 🏠 Questions Générales

### ❓ Qu'est-ce que HorizonImmo ?

**HorizonImmo** est une plateforme immobilière moderne qui connecte acheteurs, vendeurs et professionnels de l'immobilier. Elle propose des outils avancés de recherche, de gestion et de communication pour faciliter les transactions immobilières.

### 🔧 Quelles sont les principales fonctionnalités ?

-   **Recherche avancée** avec filtres géographiques et critères multiples
-   **Gestion complète des propriétés** avec galeries photos
-   **Système de messagerie** intégré entre clients et agents
-   **Interface d'administration** pour les professionnels
-   **Profils utilisateurs** avec avatars et préférences
-   **Notifications** en temps réel

### 💰 La plateforme est-elle gratuite ?

Actuellement, l'utilisation de base est gratuite pour les particuliers. Les fonctionnalités premium et l'accès professionnel sont disponibles sur abonnement. Consultez notre [page tarifaire] pour plus de détails.

### 📱 Y a-t-il une application mobile ?

La plateforme est optimisée pour mobile via notre **Progressive Web App (PWA)**. Vous pouvez l'installer sur votre smartphone depuis votre navigateur pour une expérience similaire à une application native.

## 👤 Compte Utilisateur

### 🔐 Comment créer un compte ?

1. Cliquez sur **"S'inscrire"** en haut à droite
2. Remplissez vos informations (nom, email, mot de passe)
3. Vérifiez votre email et cliquez sur le lien de confirmation
4. Votre compte est activé !

### 🔑 J'ai oublié mon mot de passe, que faire ?

1. Cliquez sur **"Mot de passe oublié"** sur la page de connexion
2. Saisissez votre adresse email
3. Consultez votre boîte mail pour le lien de réinitialisation
4. Créez un nouveau mot de passe sécurisé

### 🖼️ Comment ajouter/modifier mon avatar ?

1. Accédez à votre **profil** (menu utilisateur)
2. Section **"Photo de profil"**
3. Cliquez sur **"Télécharger un nouvel avatar"**
4. Sélectionnez une image (JPG, PNG, GIF - max 2MB)
5. Cliquez sur **"Mettre à jour l'avatar"**

### ✉️ Comment modifier mon adresse email ?

1. Allez dans votre **profil**
2. Section **"Informations personnelles"**
3. Modifiez l'adresse email
4. **Important** : Vous devrez confirmer la nouvelle adresse via email

### 🗑️ Comment supprimer mon compte ?

1. Accédez à votre **profil**
2. Section **"Suppression du compte"**
3. Saisissez votre mot de passe pour confirmation
4. Cliquez sur **"Supprimer définitivement"**
   ⚠️ **Attention** : Cette action est irréversible !

## 🏠 Recherche et Propriétés

### 🔍 Comment effectuer une recherche avancée ?

1. Utilisez la **barre de recherche** principale
2. Cliquez sur **"Filtres avancés"** pour plus d'options :
    - Type de bien (appartement, maison, terrain...)
    - Fourchette de prix
    - Nombre de pièces
    - Surface minimum/maximum
    - Localisation précise

### 📍 Comment rechercher près de chez moi ?

1. Activez la **géolocalisation** dans votre navigateur
2. Cliquez sur l'icône **"Près de moi"** dans la recherche
3. Sélectionnez le rayon de recherche (1-50 km)
4. Les résultats s'affichent automatiquement

### ❤️ Comment sauvegarder mes propriétés favorites ?

1. Sur chaque fiche propriété, cliquez sur l'icône **❤️**
2. Le bien est automatiquement ajouté à vos favoris
3. Accédez à vos favoris via le menu **"Mes Favoris"**
4. Organisez-les par listes personnalisées

### 🔔 Comment être alerté des nouveaux biens ?

1. Effectuez une recherche avec vos critères
2. Cliquez sur **"Sauvegarder cette recherche"**
3. Nommez votre alerte
4. Choisissez la fréquence (immédiate, quotidienne, hebdomadaire)

### 🗺️ Comment utiliser la carte interactive ?

1. Cliquez sur l'onglet **"Carte"** dans les résultats
2. **Navigation** : Zoom, déplacement libre
3. **Clustering** : Les biens proches sont regroupés
4. **Informations** : Survolez les marqueurs pour un aperçu
5. **Street View** : Cliquez sur les icônes pour la vue street

## 🏢 Interface Administration

### 👨‍💼 Comment devenir administrateur/agent ?

L'accès administrateur est réservé aux professionnels partenaires. Contactez notre équipe commerciale : commercial@horizonimmo.fr

### 🏠 Comment ajouter une nouvelle propriété ?

1. Interface admin → **"Propriétés"** → **"Ajouter"**
2. Remplissez les informations obligatoires :
    - Titre et description
    - Prix et localisation
    - Caractéristiques (pièces, surface...)
3. **Images** : Upload multiple (max 10 images, 2MB chacune)
4. **Options** : Bien en vedette, statut de disponibilité

### 🖼️ Quels formats d'images sont acceptés ?

-   **Formats** : JPG, JPEG, PNG, GIF
-   **Taille maximum** : 2MB par image
-   **Nombre maximum** : 10 images par propriété
-   **Dimensions recommandées** : 1200x800px minimum
-   **Compression** : Automatique lors de l'upload

### 📊 Comment accéder aux statistiques ?

1. Dashboard admin → Section **"Analytics"**
2. Métriques disponibles :
    - Nombre de vues par propriété
    - Sources de trafic
    - Taux de conversion
    - Messages reçus

### 👥 Comment gérer les utilisateurs ?

1. Menu admin **"Utilisateurs"**
2. Actions disponibles :
    - Voir tous les utilisateurs
    - Modifier les rôles (client/admin)
    - Réinitialiser mots de passe
    - Supprimer des comptes

## 🔧 Problèmes Techniques

### 🚫 Le site ne s'affiche pas correctement

1. **Vérifiez votre connexion internet**
2. **Actualiser la page** (Ctrl+F5 ou Cmd+R)
3. **Vider le cache** de votre navigateur
4. **Tester avec un autre navigateur**
5. Si le problème persiste : bugs@horizonimmo.fr

### 📸 Impossible d'uploader des images

**Vérifications :**

-   Taille du fichier < 2MB
-   Format : JPG, PNG, ou GIF
-   Connexion internet stable
-   Navigateur à jour

**Solutions :**

1. Compresser l'image avec un outil en ligne
2. Changer de format (PNG → JPG)
3. Vider le cache navigateur
4. Réessayer plus tard

### 🔄 La page se charge lentement

**Causes possibles :**

-   Connexion internet lente
-   Trop d'images en haute résolution
-   Cache navigateur plein

**Solutions :**

1. Vérifier la vitesse de connexion
2. Vider le cache navigateur
3. Fermer autres onglets/applications
4. Réessayer en navigation privée

### 📱 Problèmes sur mobile

1. **Mettre à jour** votre navigateur mobile
2. **Vider le cache** de l'application
3. **Réinstaller la PWA** si installée
4. **Tester en 4G** si problème en WiFi
5. **Redémarrer** le téléphone

### 🔐 Impossible de se connecter

**Vérifications :**

-   Email et mot de passe corrects
-   Compte activé (vérification email)
-   Pas de typos dans l'email

**Solutions :**

1. Réinitialiser le mot de passe
2. Vérifier les spams pour l'email de confirmation
3. Contacter le support : support@horizonimmo.fr

## 🔒 Sécurité et Confidentialité

### 🛡️ Mes données sont-elles sécurisées ?

**Oui, nous appliquons les meilleures pratiques :**

-   **Chiffrement HTTPS** sur tout le site
-   **Mots de passe hashés** (non stockés en clair)
-   **Conformité RGPD** complète
-   **Audits sécurité** réguliers
-   **Sauvegardes** automatiques

### 📋 Quelles données collectez-vous ?

**Données minimales nécessaires :**

-   Nom, email pour la création de compte
-   Historique de navigation (anonymisé)
-   Préférences de recherche
-   Communications volontaires

**Nous ne collectons JAMAIS :**

-   Données bancaires
-   Mots de passe en clair
-   Informations non nécessaires au service

### 🗑️ Comment supprimer mes données ?

**Droit à l'oubli RGPD :**

1. Suppression de compte → Données automatiquement effacées
2. Demande spécifique → contact@horizonimmo.fr
3. Délai légal → 30 jours maximum
4. Confirmation → Email de validation envoyé

### 👀 Qui peut voir mes informations ?

-   **Profil public** : Nom d'affichage uniquement
-   **Agents/Admins** : Coordonnées si vous les contactez
-   **Support** : Accès technique limité si problème
-   **Tiers** : Aucun partage commercial

### 🔐 Comment créer un mot de passe sécurisé ?

**Recommandations :**

-   **8 caractères minimum**
-   **Mélange** : lettres, chiffres, symboles
-   **Éviter** : mots de dictionnaire, infos personnelles
-   **Unique** : Différent pour chaque site
-   **Gestionnaire** : Utiliser un gestionnaire de mots de passe

## 📱 Mobile et Progressive Web App

### 📲 Comment installer l'app sur mon téléphone ?

**iPhone (Safari) :**

1. Visitez horizonimmo.fr
2. Icône **"Partager"** en bas
3. **"Sur l'écran d'accueil"**
4. Nommer l'app et confirmer

**Android (Chrome) :**

1. Visitez horizonimmo.fr
2. Menu **"⋮"** → **"Ajouter à l'écran d'accueil"**

### 📶 L'app fonctionne-t-elle hors ligne ?

**Fonctionnalités offline :**

-   Consultation des propriétés déjà visitées
-   Navigation dans vos favoris
-   Formulaires en mode brouillon
-   Synchronisation automatique à la reconnexion

**Nécessite une connexion :**

-   Nouvelles recherches
-   Envoi de messages
-   Upload d'images
-   Mise à jour des données

### 🔄 Comment mettre à jour l'app ?

La PWA se met à jour automatiquement. Si vous rencontrez des problèmes :

1. **Actualiser** la page dans le navigateur
2. **Réinstaller** l'app depuis le navigateur
3. **Vider le cache** de l'app
